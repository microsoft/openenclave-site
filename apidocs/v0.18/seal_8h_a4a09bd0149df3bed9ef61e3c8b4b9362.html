oe_result_t oe_register_seal_plugin ( const oe_seal_plugin_definition_t *  plugin,
bool  make_default 
)

Register a plug-in to be used by oe_seal() and oe_unseal().

Parameters
[in]pluginPointer to the plug-in being registered.
[in]make_defaultTRUE to make this plug-in the default plug-in. A registered plug-in could be made default by registering it again with make_default set to TRUE, but a default plug-in cannot be made non-default by setting make_default to FALSE.
Return values
OE_OKplugin was registered successfully.
OE_INVALID_PARAMETERAt least one parameter is invalid.
OE_OUT_OF_MEMORYToo many plug-ins have been registered.
